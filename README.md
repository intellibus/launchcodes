## Usage 🔭

Read more about the [Design](https://github.com/intellibus/launchcodes/blob/main/DESIGN.md) behind `launchcodes` here.

`.releasesrc.json`

```json
{
  "plugins": [
    [
      "launchcodes",
      {
        "projectType": "google-add-ons"
      }
    ]
  ]
}
```
